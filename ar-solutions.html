import ar_engine

# Initialize spatial environment
env = ar_engine.Environment(api_key="YOUR_KEY")

# Load 3D Asset
model = env.load_model("s3://bucket/engine_cad.gltf")

# Deploy to connected headsets
env.broadcast(
    asset=model,
    scale=1.0,
    physics=True,
    target_users=["tech_team_alpha"]
)

Integration FAQ

Can I run the software on a closed network?

Yes, our Enterprise tier includes a Dockerized on-premise version that requires zero external internet connectivity.

Does it support legacy CAD software?

We provide automated pipelines to convert proprietary CAD files (SolidWorks, AutoCAD, CATIA) into optimized spatial models automatically.

What is the minimum bandwidth required for Cloud Render?

For stable 60FPS cloud streaming of massive models, a stable 25Mbps connection via 5G or Wi-Fi 6 is recommended.
